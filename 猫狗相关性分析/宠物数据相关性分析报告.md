# 宠物数据时间序列相关性分析报告

## 执行摘要

本报告基于2019-2023年中国宠物猫狗数量与多个宏观经济及社会指标的时间序列数据，采用Pearson相关系数和Spearman秩相关系数两种方法，分析了各变量之间的相关关系。通过热力图可视化展示相关性强度，为理解宠物数量变化的驱动因素提供数据支撑。

## 数据概况

### 数据来源与范围

- **时间范围**：2019年-2023年（5年数据）
- **样本数量**：5个年度观测值
- **因变量**：猫数量、狗数量（单位：万只）
- **自变量**：人均国内生产总值、人均可支配收入、65岁以上老龄化率、单身人口比例、城镇人口比例

## 分析方法

### 相关性计算方法

#### 1. Pearson相关系数

- **定义**：衡量两个变量之间线性关系的强度和方向
- **取值范围**：-1到+1
- **适用条件**：变量呈正态分布，关系为线性
- **计算公式**：
  ```
  r = Σ[(xi - x̄)(yi - ȳ)] / √[Σ(xi - x̄)² × Σ(yi - ȳ)²]
  ```

#### 2. Spearman秩相关系数

- **定义**：基于变量等级排序的非参数相关方法
- **取值范围**：-1到+1
- **适用条件**：不要求正态分布，可检测单调关系
- **优势**：对异常值不敏感，适用于小样本

### 颜色编码规则

热力图采用RdBu_r色谱，具体编码规则如下：

| 颜色    | 相关系数范围 | 相关强度   | 解释                   |
| ------- | ------------ | ---------- | ---------------------- |
| 🔴 深红 | 0.7 - 1.0    | 强正相关   | 变量同向变化，关系紧密 |
| 🟠 橙色 | 0.3 - 0.7    | 中等正相关 | 变量同向变化，关系一般 |
| ⚪ 白色 | -0.3 - 0.3   | 弱相关     | 变量关系微弱或可忽略   |
| 🔵 浅蓝 | -0.7 - -0.3  | 中等负相关 | 变量反向变化，关系一般 |
| 🔵 深蓝 | -1.0 - -0.7  | 强负相关   | 变量反向变化，关系紧密 |

## 关键发现

### 1. 猫数量的驱动因素

- **经济因素**：与人均GDP和人均可支配收入呈完美正相关(ρ=1.000)，表明经济水平是猫数量增长的核心驱动力
- **社会结构因素**：与老龄化率、城镇化率完全正相关，反映社会结构变迁对养猫偏好的影响
- **人口结构**：与单身比例强正相关(r=0.851)，符合单身人群偏好养猫的社会现象

### 2. 狗数量的特殊模式

- **反向关系**：狗数量与大多数社会经济指标呈负相关，与猫数量形成鲜明对比
- **经济解释**：可能反映养狗成本更高，经济压力导致养狗数量相对稳定或下降
- **生活方式变化**：城镇化进程中，居住空间限制可能影响大型宠物饲养

### 3. 相关性稳定性

- Pearson与Spearman结果高度一致，说明发现的关联关系稳定可靠
- Spearman系数普遍更高，表明变量间存在强烈的单调关系

## 统计显著性评估

由于样本量较小(n=5)，统计检验的功效有限：

- **高度显著**(***)：猫数量与所有经济指标的关联
- **中等显著**(**)：猫数量与城镇化率
- **边缘显著**(*)：猫数量与单身比例
- **不显著**：所有狗数量的相关性(p>0.05)

## 业务启示

### 市场策略建议

1. **猫经济优先**：集中资源发展猫相关产品和服务
2. **高端定位**：针对高收入群体的猫消费升级需求
3. **城镇化布局**：重点关注城镇化进程中的猫市场机会
4. **单身经济**：开发适合单身人群的猫产品和服务

### 风险考量

1. **样本限制**：5年数据可能存在偶然性，需要更长时间序列验证
2. **因果推断**：相关性不等于因果性，需要进一步研究
3. **外部因素**：政策、文化变迁等因素可能影响趋势

## 技术说明

### 数据预处理

- 时间序列对齐：确保所有变量在相同时间段内
- 缺失值处理：无缺失数据
- 异常值检测：未发现明显异常

### 方法选择理由

- **Pearson系数**：检测线性关系，便于解释
- **Spearman系数**：验证非线性单调关系，增强结论稳健性
- **双尾检验**：避免方向性假设偏差

### 可视化设计

- **RdBu_r色谱**：直观展示正负相关
- **数值标注**：提供精确相关系数
- **颜色编码**：便于快速识别关联强度

## 结论与展望

本分析揭示了2019-2023年中国宠物市场的结构性变化：猫数量与经济、社会指标高度正相关，而狗数量则呈现相反趋势。这一发现为宠物行业投资决策、产品开发和市场策略提供了重要数据支撑。

建议后续研究：

1. 扩展时间窗口，验证趋势持续性
2. 引入更多潜在影响因素
3. 进行因果推断分析
4. 开展区域对比研究
