# 中国宠物数量ARIMAX时间序列预测建模分析报告

## 执行摘要

本报告基于2017-2023年中国宠物数量历史数据，采用ARIMAX（自回归积分滑动平均模型）方法对未来三年（2024-2026年）猫狗数量进行预测分析。通过系统性的数据预处理、参数优化、模型构建和验证，建立了两个独立的ARIMAX预测模型，为宠物行业发展趋势提供量化分析支撑。

### 主要发现
- **猫数量预测**：2024-2026年预测值分别为10,165万只、11,665万只、10,880万只
- **狗数量预测**：2024-2026年预测值分别为16,094万只、16,262万只、16,430万只  
- **模型精度**：猫模型AIC=106.04，狗模型AIC=82.74，均显示良好的模型拟合效果
- **增长趋势**：猫数量呈现波动性增长，狗数量保持相对稳定增长

## 1. 研究背景与目标

### 1.1 研究背景
随着中国经济发展和居民生活水平提升，宠物行业快速发展。准确预测宠物数量变化趋势对于行业规划、投资决策、政策制定具有重要意义。传统时间序列模型往往只考虑历史数据本身，而ARIMAX模型能够纳入外部经济和社会因素，提高预测精度。

### 1.2 研究目标
1. 构建基于多变量时间序列的ARIMAX预测模型
2. 预测2024-2026年中国猫狗数量变化趋势
3. 评估模型性能和预测不确定性
4. 为相关决策提供量化分析支撑

## 2. 数据与方法

### 2.1 数据来源
- **时间跨度**：2017-2023年（7年历史数据）
- **核心变量**：猫数量、狗数量（因变量）
- **外生变量**：养宠家庭渗透率、人均GDP、人均可支配收入、城市化率、老龄化率
- **数据质量**：数据完整性100%，无缺失值

### 2.2 分析方法
采用ARIMAX（Autoregressive Integrated Moving Average with eXogenous variables）模型，该方法结合了传统ARIMA时间序列分析和多元回归分析的优势。

**模型公式**：
```
(1 - φ₁L - φ₂L²)(1 - L)Y_t = c + (1 + θ₁L)ε_t + β₁X₁t + β₂X₂t + β₃X₃t
```

其中：
- Y_t：时间t的宠物数量（猫/狗）
- L：滞后算子
- φ：自回归系数
- θ：移动平均系数  
- X_it：外生变量（经济指标等）
- β_i：外生变量系数
- ε_t：白噪声误差项

## 3. 数据预处理与探索性分析

### 3.1 数据平稳性检验
采用ADF（Augmented Dickey-Fuller）检验对时间序列进行平稳性分析：

| 变量 | 原始序列 | 一阶差分 | 二阶差分 | 结论 |
|------|----------|----------|----------|------|
| 猫数量 | 非平稳 | 平稳 | - | I(1) |
| 狗数量 | 非平稳 | 平稳 | - | I(1) |
| 养宠渗透率 | 平稳 | - | - | I(0) |
| 人均GDP | 非平稳 | 平稳 | - | I(1) |
| 城市化率 | 非平稳 | 非平稳 | 平稳 | I(2) |

### 3.2 相关性分析
通过Pearson和Spearman相关性分析识别关键预测因子：

**猫数量相关性**（|r| > 0.9）：
- 养宠家庭渗透率：r = 0.994（极强正相关）
- 人均GDP：r = 0.983（极强正相关）
- 人均可支配收入：r = 0.974（极强正相关）

**狗数量相关性**（|r| > 0.5）：
- 养宠家庭渗透率：r = 0.823（强正相关）
- 城市化率：r = 0.567（中等正相关）

### 3.3 异常值检测
采用IQR（四分位距）方法进行异常值检测，结果显示所有变量均在正常范围内，无异常值需要处理。

## 4. 模型构建与参数优化

### 4.1 参数选择策略
采用网格搜索（Grid Search）方法，在以下参数空间内寻找最优组合：
- p（自回归阶数）：0-3
- d（差分阶数）：1（基于平稳性检验）
- q（移动平均阶数）：0-2

### 4.2 最优参数确定
通过AIC（Akaike信息准则）和BIC（Bayesian信息准则）进行模型选择：

**猫数量ARIMAX模型**：ARIMAX(2,1,1)
- AIC = 106.04，BIC = 104.58
- 外生变量：养宠渗透率、人均GDP、人均可支配收入
- 对数似然值 = -46.02

**狗数量ARIMAX模型**：ARIMAX(0,1,1)
- AIC = 82.74，BIC = 81.90  
- 外生变量：养宠渗透率、城市化率
- 对数似然值 = -37.37

### 4.3 模型验证
对模型残差进行诊断检验：

**残差白噪声检验**（Ljung-Box检验）：
- 猫模型：p值 > 0.05，残差为白噪声
- 狗模型：p值 > 0.05，残差为白噪声

**残差正态性检验**（Jarque-Bera检验）：
- 猫模型：p值 = 0.169，残差服从正态分布
- 狗模型：p值 = 0.409，残差服从正态分布

## 5. 预测结果与分析

### 5.1 数量预测结果

**猫数量预测（万只）**：
| 年份 | 预测值 | 年增长率 | 95%置信区间 |
|------|--------|----------|-------------|
| 2024 | 10,165 | -0.3% | [9,851 - 10,479] |
| 2025 | 11,665 | 14.8% | [11,351 - 11,979] |
| 2026 | 10,880 | -6.7% | [10,566 - 11,194] |

**狗数量预测（万只）**：
| 年份 | 预测值 | 年增长率 | 95%置信区间 |
|------|--------|----------|-------------|
| 2024 | 16,094 | 0.6% | [13,099 - 19,089] |
| 2025 | 16,262 | 1.0% | [13,267 - 19,257] |
| 2026 | 16,430 | 1.0% | [13,435 - 19,425] |

### 5.2 趋势分析

**猫数量趋势特征**：
1. **短期波动**：2024年出现小幅回调（-0.3%）
2. **中期爆发**：2025年大幅增长14.8%，达到预测峰值
3. **后期调整**：2026年回落6.7%，但仍高于2024年水平
4. **总体趋势**：呈现"N型"增长模式，反映市场成熟度提升过程中的调整

**狗数量趋势特征**：
1. **稳定增长**：保持年均1%左右的温和增长
2. **市场成熟**：增长率明显低于猫数量，显示市场趋于饱和
3. **持续上升**：连续三年正增长，但增幅有限
4. **总体趋势**：呈现平稳增长态势，符合成熟市场特征

### 5.3 预测不确定性分析

基于模型残差标准差计算95%置信区间：
- **猫数量**：预测标准误差约±320万只
- **狗数量**：预测标准误差约±5,870万只

狗数量预测区间较宽，反映其受更多复杂因素影响，预测不确定性相对较高。

## 6. 模型性能评估

### 6.1 拟合优度指标

| 模型 | AIC | BIC | 对数似然值 | 样本量 |
|------|-----|-----|------------|--------|
| 猫ARIMAX(2,1,1) | 106.04 | 104.58 | -46.02 | 7 |
| 狗ARIMAX(0,1,1) | 82.74 | 81.90 | -37.37 | 7 |

两个模型的AIC和BIC值均相对较小，表明模型在复杂度和拟合优度之间取得了良好平衡。

### 6.2 残差分析

**猫模型残差统计**：
- 均值：-112.75万只（接近0，无系统性偏差）
- 标准差：160.35万只（预测精度较高）
- 分布特征：基本服从正态分布

**狗模型残差统计**：
- 均值：1,162.28万只（存在一定偏差）
- 标准差：2,995.00万只（预测波动性较大）
- 分布特征：正态性假设基本满足

### 6.3 模型比较

猫模型表现优于狗模型：
1. 残差标准差更小（160 vs 2,995万只）
2. 系统性偏差更小（-113 vs +1,162万只）
3. 预测精度更高，不确定性更低

## 8. 模型局限性与改进建议

### 8.1 模型局限性

**数据限制**：
1. 样本量较小（仅7年历史数据），影响参数估计稳定性
2. 缺乏更长时间跨度的历史数据进行模型验证
3. 外生变量选择受限于可获得数据

**模型假设**：
1. 线性关系假设可能无法捕捉复杂的非线性动态
2. 结构性变化（如疫情冲击）未充分考虑
3. 预测期外生变量假设基于简单趋势外推

**预测不确定性**：
1. 狗数量预测不确定性显著高于猫数量
2. 长期预测（2026年）不确定性较大
3. 置信区间宽度反映模型精度有限

### 8.2 改进建议

**数据层面**：
1. 收集更长时间序列数据（建议10年以上）
2. 增加更多潜在影响因素（如政策变化、消费偏好等）
3. 考虑高频数据（月度或季度数据）提高模型精度

**模型层面**：
1. 探索非线性时间序列模型（如神经网络、支持向量机）
2. 考虑状态空间模型和卡尔曼滤波方法
3. 引入贝叶斯方法处理参数不确定性

**应用层面**：
1. 建立滚动预测机制，定期更新模型参数
2. 结合专家判断和定性分析
3. 开发情景分析，考虑不同经济环境下的预测结果

## 9. 结论与政策建议

### 9.1 主要结论

1. **市场分化明显**：猫数量呈现波动性增长，狗数量保持平稳增长，反映两个细分市场的不同发展阶段

2. **经济因素影响**：人均GDP、可支配收入等经济指标对猫数量影响更为显著，说明猫消费对经济环境敏感度更高

3. **预测精度差异**：猫数量预测模型表现优于狗数量模型，可能与狗市场成熟度更高、影响因素更复杂有关

4. **增长空间评估**：猫市场仍有较大增长潜力，狗市场趋于饱和，建议行业资源向猫相关产品和服务倾斜

### 9.2 政策建议

**产业政策**：
1. 支持猫产业链发展，包括食品、医疗、用品等细分领域
2. 鼓励创新产品和服务，满足消费者多样化需求
3. 建立行业标准，规范市场有序发展

**投资方向**：
1. 重点关注猫经济相关投资机会
2. 布局高端宠物服务和产品
3. 关注二三线城市市场潜力

**风险管控**：
1. 建立市场监测预警机制
2. 定期更新预测模型和参数
3. 制定应对市场波动的策略预案
